# Regras de trabalho do projeto

- Sempre que houver alteração em arquivos do projeto, subir para o GitHub.
- Para a telemetria GT7, usar a branch `telemetria-4.0` quando o pedido for da V4.
- Entregar APK quando for possível gerar ou reaproveitar o pacote com a interface atualizada.
- Não mexer no Raspberry sem pedido explícito.
